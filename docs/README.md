1. 컴퓨터가 임의의 서로 다른 3자리의 수 생성


2. 유저로 부터 3자리의 서로 다른 숫자 입력 받음


3. 입력 값 검증, 아니면 IllegalArgumentException 발생 

   3-1) 3자리가 맞는지 확인

   3-2) 숫자가 맞는지 확인

   3-3) 그 3자리 숫자가 서로 다른지 확인


4. 정답과 비교 후 결과 출력, 반복

   4-1) 스트라이크, 볼 수 세기

   4-2) 정답(3 스트라이크)인지 체크



5. 정답일 시, 입력값(1 or 2) 에 따라 따라 재시작 혹은 종료

   5-1) 입력값(1 or 2) 검증
